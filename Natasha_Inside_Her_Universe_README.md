# 🌌 Natasha’s Interactive Birthday Experience
## “Inside Her Universe”

A cinematic, surreal, interactive birthday experience built as a **single-file HTML application**.

---

## ✨ Overview

This project is a **10-scene interactive journey** designed to feel like stepping inside Natasha’s universe.

It is not a traditional story.

It is:
- Abstract  
- Emotional  
- Symbolic  
- Experiential  

Each scene represents a **metaphor**, not a moment.

---

## 🎮 Experience Structure

The experience unfolds across **10 interactive scenes + 1 final reveal**:

1. 🌑 The Void — light emerges from nothing  
2. 🌌 Gravity Shift — world becomes unstable  
3. 🎨 Color Awakening — color returns  
4. 🧩 Imperfect Puzzle — broken pieces connect  
5. ⏳ Time Loop — repetition breaks  
6. 🎭 Multiple Faces — shifting emotions  
7. 🕯️ Light Reveal — truth appears  
8. 🌀 Chaos System — order fails, beauty remains  
9. 🌊 Ripple World — impact spreads  
10. 🎁 Reality Break — illusion shatters  

Final Scene:
💖 Emotional reveal + birthday message
